Descargar

Información de la última versión:

La imagen ISO de Arch Linux puede ser grabada a un CD o copiarse directamente vía ‘dd‘ hacia un Pendrive o disco USB booteable. Está imagen es para instalaciones nuevas o para utilizarla como entorno “Live” (booteo directo) en casos de reparación.

Si ya sos usuario de Arch Linux y deseas actualizar tu sistema, deberías utilizar pacman -Syu

Si tenes dudas o necesitas ayuda, tanto para instalar o actualizar, consulta en el foro.

Versión ISO actual:  2019.08.01
Kernel: 5.2.5
Tamaño de la ISO: 622.0 MB
Guía de Instalación

Descargar vía BitTorrent:

Para descargar Arch Linux y colaborar con los servidores de toda la comunidad, recomendamos hacerlo vía Bittorrent. El enlace siguiente es para el torrent de ambas arquitecturas; x86 y x86-64 (32 y 64bits respectivamente). Otros métodos de descarga:

Si deseas buscar otras opciones de descarga (FTP, HTTP) visita el sitio oficial de Arch Linux en inglés para obtener el último enlace de descarga oficial. En lo posible recorda utilizar Bittorrent para colaborar con la comunidad y reducir los gastos de ancho de banda. Instalación:

Una vez que hayas descargado podes seguir los pasos de la Wiki en español para instalarlo. Si te surgen dudas o queres compartir algún comentario podes visitar los foros en español de Arch Linux. Allí encontraras el foro de instalación, entre otros.

Vagrant

Hay imagenes oficiales de Arch Linux para Vagrant con libvirt y virtualbox en Vagrant Cloud. Tambien pueden hacer un boostrap de la imagen de vagrant:

vagrant init archlinux/archlinux
vagrant up

Docker

Hay una imagen oficial de Docker de Arch Linux que pueden encontrar en Docker Hub. O correr la imagen con el siguiente comando.

docker run -it archlinux/base
